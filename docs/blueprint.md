# **App Name**: IDMC CogniAssistant

## Core Features:

- AI-Powered IDMC Question Answering: Utilize generative AI to understand user questions about Informatica Data Management Cloud (IDMC) and provide accurate, contextually relevant answers.
- Conversational Interface: A user-friendly chat interface for seamless interaction, allowing users to ask questions and receive AI-generated responses easily.
- Multi-Model Orchestration Tool: An intelligent tool that integrates and orchestrates responses from various advanced AI models (such as GPT, Gemini, and Copilot) to generate the most comprehensive and relevant answer.
- Dynamic Knowledge Retrieval Tool: A tool that proactively searches and synthesizes information from extensive IDMC documentation and resources, using it to ground AI responses and ensure factual accuracy.
- Interactive Answer Display: Present AI-generated answers clearly and concisely, potentially including relevant deep links to official IDMC documentation for further reference.
- Query History Management: Allow users to review their past questions and the corresponding AI responses, enabling easy access to previously gathered information.

## Style Guidelines:

- Primary brand color: A deep, intelligent indigo (#361FAD), evoking sophistication and data mastery.
- Background color: A very subtle, cool-toned off-white (#F1F0F5), providing a clean and professional canvas that highlights content.
- Accent color: A vibrant, clear blue (#5E8FEBS), used to draw attention to key interactive elements and provide visual dynamism.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and crisp aesthetic that complements the AI theme.
- Body font: 'Inter' (sans-serif) for exceptional readability across all screen sizes and content lengths, ensuring a clear and objective user experience.
- Employ a set of clear, minimalist line-art icons that intuitively represent actions like 'send message,' 'history,' 'settings,' and 'information,' maintaining a clean and professional look.
- Implement a clean, spacious layout focused on the conversational exchange, using generous white space to enhance readability and reduce cognitive load for users interacting with AI responses.
- Utilize subtle and functional micro-animations for feedback, such as typing indicators, message loading, and state changes, providing a responsive and fluid user experience without distraction.
